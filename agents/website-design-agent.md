## OUTPUT FORMAT

Each design variant includes:

1. **Style Guide Document**
   ```markdown
   # [Variant Name] Style Guide
   
   ## Colors
   - Primary: #hex (name)
   - Secondary: #hex (name)
   - Accent: #hex (name)
   
   ## Typography
   - Display: Font Family, weights
   - Body: Font Family, weights
   
   ## Spacing
   - Base: 8px
   - Section padding: 80px (desktop), 40px (mobile)
   
   ## Border Radius
   - Small: 8px
   - Medium: 16px
   - Large: 24px
   
   ## Shadows
   - Subtle: 0 4px 12px rgba(0,0,0,0.1)
   - Elevated: 0 8px 24px rgba(0,0,0,0.15)
   ```

2. **Component Library (Markdown)**
   ```markdown
   ## Buttons
   **Primary:**
   - Background: #hex gradient
   - Color: #hex
   - Padding: 16px 32px
   - Hover: translateY(-2px) + shadow
   
   **Secondary:**
   - Border: 2px solid #hex
   - Background: transparent
   - Hover: background #hex
   
   ## Form Fields
   - Border: 1px solid #hex
   - Focus: border #accent + glow
   - Error: border red + shake
   ```

3. **Layout Wireframes (ASCII/Description)**
   ```
   [HERO - Full-screen with background image]
   [SERVICES - 3-column grid with cards]
   [GALLERY - Bento grid with hover effects]
   [FOOTER - 3 columns]
   ```

4. **Animation Specifications**
   ```javascript
   // Hero text stagger
   gsap.from('.hero h1 span', {
     y: 100,
     opacity: 0,
     stagger: 0.1,
     duration: 1,
     ease: 'power4.out'
   });
   ```
